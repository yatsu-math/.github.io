【問題】

3． $\triangle ABC$ を鋭角三角形とし，$AB > AC$ とする。$\triangle ABC$ の外接円の $B$ と $C$ における接線の交点を $P$ とする。線分 $PB$ と $PC$ の中点を通る直線は，直線 $AB$ と $AC$ とそれぞれ $X$ と $Y$ で交わる。四角形 $AXPY$ が円に内接することを証明せよ。
解説を表示する
【解説の図形（ここに幾何図版を配置）】

$\triangle ABC$ の外接円を円 $O$ とし，中心を $O$ とする。

線分 $PB$，$PC$ の中点をそれぞれ $M$，$N$ とすると，$\triangle BPC$ で中点連結定理より，$BC \parallel MN$ だから，$BC \parallel XY$ である。また，点 $P$ を通り $BC$ と平行な直線と直線 $AB$，$AC$ との交点をそれぞれ $D$，$E$ とする（…①）。

すると $\triangle BDP$ において，$BM = MP$ であるから中点連結定理より点 $X$ は線分 $BD$ の中点であり，同様に点 $Y$ は線分 $CE$ の中点である。

$\angle CAB = a$，$\angle ABC = b$，$\angle BCA = c$ とすると，$a + b + c = 180^\circ$（…②）。円 $O$ と直線 $BP$，$CP$ についてそれぞれの接弦定理から，$\angle CBP = \angle BCP = a$ だから、②より $\angle PBD = c$，$\angle PCE = b$ である。また①より $\angle ADE = b$ だから，$\triangle BDP$ の内角の和から，$\angle BPD = a$。同様にして，$\angle CEP = c$，$\angle CPE = a$ となる。このことにより，$$\triangle ABC \sim \triangle PDB \sim \triangle PCE$$ である。

そこで辺 $BC$ の中点を $L$ として，$\angle BAL = p$，$\angle CAL = q$ とする。すると $\triangle PDB$ において点 $X$ は辺 $DB$ の中点だから，$\triangle ABL \sim \triangle PDX$ となるから，$\angle DPX = p$。同様にして $\triangle EPY = \angle CAL = q$ となる。

すると，$\angle XAY = p + q$，$\angle XPY = 180^\circ - (p + q)$ だから，四角形 $AXPY$ は円に内接する。
